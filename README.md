# WorldExplorer - Dunyo Davlatlari Ma'lumotlar Platformasi

🌍 **WorldExplorer** — bu butun dunyo davlatlari haqida to'liq va aniq ma'lumot beruvchi zamonaviy veb-ilova. Ushbu loyiha yordamida har bir davlatning aholisi, hududi, poytaxti, bayrog'i, chegaradosh davlatlari va boshqa ko'plab qiziqarli ma'lumotlarni topishingiz mumkin.

## 🚀 Texnologiyalar Steki

Loyiha eng zamonaviy texnologiyalar yordamida qurilgan:
* **Framework:** Next.js 15 (App Router)
* **Til:** TypeScript
* **Dizayn va Stil:** Tailwind CSS
* **Ma'lumot manbasi:** REST Countries API
* **Holatni saqlash:** LocalStorage (Sevimlilar uchun)
* **Deploy:** Vercel

## ✨ Asosiy Imkoniyatlar

* **Davlatlarni qidirish:** Istalgan davlatni nomi orqali tezkor qidirish (Debounce integratsiyasi bilan).
* **Mintaqalar bo'yicha filtrlash:** Davlatlarni qit'alar bo'yicha ajratib ko'rish.
* **Saralash:** Alifbo, aholi soni yoki hudud maydoni bo'yicha tizimlashtirilgan saralash imkoniyati.
* **Batafsil ma'lumot sahifasi:** Har bir davlat uchun alohida, chuqurlashtirilgan va barcha tafsilotlarni o'z ichiga oluvchi sahifa (poytaxt, tillar, valyuta, geografik joylashuv va chegaradosh davlatlar).
* **Sevimlilar:** Yoqtirgan davlatlarni saqlab qo'yish tizimi (ma'lumotlar brauzer xotirasida saqlanadi).
* **Responsive tizim:** Ilova barcha qurilmalarda - mobil, planshet va kompyuterlarda tushunarli hamda chiroyli ishlaydi.

## 🛠 O'rnatish va Ishga tushirish

1. Repozitoriyni nusxalang:
   ```bash
   git clone https://github.com/sanjarbek404/World-Explorer.git
   cd world-explorer
   ```

2. Kerakli paketlarni o'rnating:
   ```bash
   npm install
   ```

3. Dasturni ishlab chiquvchi holatida (development) ishga tushiring:
   ```bash
   npm run dev
   ```

4. Brauzeringizda `http://localhost:3000` manzilini oching.
